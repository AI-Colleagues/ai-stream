# Smart Stream

A smart app build with OpenAI and Streamlit. The needed Streamlit widgets will be rendered according to your requests.

## Set up

Install:
```
pip install -e .
```

To set up, add `OPENAI_API_KEY` in your `.env` file.

## Run

`streamlit run smart_stream/app.py`
